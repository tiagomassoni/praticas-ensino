Principal

Informações acadêmicas com clareza e leveza.

Este site apresenta um ponto de apoio para alunos e docentes acompanharem orientações, avisos e materiais de forma simples. A proposta é oferecer uma base organizada, direta e fácil de atualizar com o tempo.

Sobre

Uma base inicial para comunicação com estudantes e docentes

A primeira versão foi pensada para ser objetiva e profissional. Com poucos arquivos e uma organização simples, o site já pode servir como página pública de referência no GitHub Pages.

Objetivo

Centralizar informações introdutórias, textos de apoio e comunicados de maneira acessível para diferentes públicos da comunidade acadêmica.

Manutenção

Todo o conteúdo pode ser ajustado diretamente no HTML, enquanto o visual permanece concentrado em um único arquivo CSS para facilitar revisões futuras.
